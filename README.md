**System Overview: 1C Fitness Club Database**

The database includes a client directory and an information register for client management, where each record contains only the client’s name.

In the "Memberships and Visits" document, you can:  
- Select a client  
- Add memberships via the tabular section    

*Price is filled in automatically upon selection.*

### Additional Features in the Tabular Section:

"Brought a Friend" (Boolean flag)  
→ If true, a 15% discount is applied to the membership cost.

Automatic End Date Calculation  
→ End date is calculated automatically for:  
 • Weekly  
 • Monthly  
 • Annual memberships  
→ For "Single Visit", the "End Date" field is hidden entirely.*
